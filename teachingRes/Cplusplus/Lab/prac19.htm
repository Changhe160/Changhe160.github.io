实验十九 输入/输出与重载

一、 实验目的

1、 掌握输入、输出的格式控制。理解类ios提供的公有的无名枚举类型的格式控制符及怎样使用类ios的公有接口函数实现控制。

2、 掌握流操作子

3、 熟悉标准设备的输入/输出。特别是cin的使用细则，提高输入的稳健性。

4、 掌握输入/输出运算符的重载。

二、 实验内容

1、 编程：按下列要求实现数据的输入/输出：

a）分别以左和右对齐方式输出整数，域宽为12；

b) 以八进制、十进制、十六进制输入输出整数；

c) 实现浮点数的指数格式和定点格式的输入输出，并指定精度，按你所见到的实际情况进行调整，直到满意；

d) 把字符串读入字符型数组变量中，从键盘输入，要求输入串的空格也全部读入，以回车换行符结束；

e) 以上要求用流成员函数和流操作子各做一遍。

2、 范例：从输入流中分析出数字串。

#include< iostream>

using namespace std;

int main(){

int getnum(char *);

char buf[100];

while (getnum(buf)){

cout << "Digit string is :";

cout << buf << endl;

}

return 0;

}

int getnum(char *s){

*s = '\0';

char ch;

while (cin.get(ch) && !cin.eof() && !isdigit(ch)) ;//去掉前面的非数字字符

do //读入数字串

*s++ = ch;

while (cin.get(ch) && !cin.eof() && isdigit(ch));

*s = '\0';，

if (!cin.eof()) cin.putback(ch); //A

if (!cin || cin.eof()) return 0; //B

return 1;

}
